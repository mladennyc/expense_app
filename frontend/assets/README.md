# KASA App Assets

This folder contains all the visual assets for the KASA app.

## Required Files

- `icon.png` - Main app icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `splash.png` - Splash screen image (1242x2436px recommended)
- `favicon.png` - Web favicon (48x48px or 32x32px)

See [LOGO_SETUP.md](./LOGO_SETUP.md) for detailed instructions on creating and adding logos.
